# Foodgram онлайн-платформа для хранения и обмена рецептами


**Цель проекта** — предоставить пользователям удобную онлайн-платформу для создания, хранения и обмена рецептами.

### Возможности сайта

-  Создание и хранение собственных рецептов
-  Скачивание списка продуктов для приготовления блюда
-  Просмотр рецептов друзей
-  Добавление любимых рецептов в избранное

> Для доступа ко всем функциям сайта необходима регистрация.  
> Проверка email не осуществляется — вы можете использовать любой адрес.

**Заходите и делитесь своими любимыми рецептами!**

## Технологии

### Backend

- **Python 3.9** — основной язык программирования
- **Django** — фреймворк для разработки веб-приложений
- **Django REST Framework** — создание RESTful API
- **Djoser** — управление аутентификацией пользователей через REST API
- **Gunicorn** — WSGI HTTP сервер для запуска Django-приложения

### Frontend

- **React** — фреймворк для создания одностраничных приложений (SPA)

### Инфраструктура и развертывание

- **Docker** — контейнеризация приложения (отдельные контейнеры для backend,db, frontend и nginx)
- **Nginx** — веб-сервер и обратный прокси
- **PostgreSQL** — реляционная база данных

### CI/CD и автоматизация

- **GitHub Actions** — автоматизация тестирования и деплоя (CI/CD)
- **Ubuntu** — операционная система сервера для развертывания приложения
